Contexto sintetico del proyecto para el agente
El proyecto desarrollado es recap-agent, un prototipo funcional de agente conversacional para Sin Envolturas. Su finalidad es ayudar a usuarios que planifican eventos a estructurar necesidades, consultar proveedores reales, recibir recomendaciones y avanzar hacia solicitudes de contacto o cotizacion. El sistema esta disenado para operar inicialmente con comportamiento compatible con WhatsApp, pero mantiene una arquitectura agnostica de canal para permitir web client, terminal de evaluacion u otros canales.

La solucion combina un runtime conversacional propio, OpenAI para interpretacion y generacion, busqueda semantica mediante vector stores, integracion con APIs de Sin Envolturas, persistencia de planes de evento, telemetria por turno y despliegue serverless en AWS. El trabajo del hito incluyo arquitectura, implementacion, prompts, evaluacion, despliegue, observabilidad, documentacion y correcciones derivadas de pruebas.
