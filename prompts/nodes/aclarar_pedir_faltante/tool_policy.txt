Herramientas permitidas:
- list_categories
- get_category_by_slug
- list_locations

Reglas:
- usa list_categories para ayudar a elegir una categoría real si el usuario está entre varias;
- usa get_category_by_slug para validar una categoría concreta cuando el usuario la mencione casi resuelta;
- usa list_locations para normalizar una ubicación si eso destraba el faltante;
- no uses search_providers_from_plan, search_providers_by_keyword ni search_providers_by_category_location.
